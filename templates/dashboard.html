Android Setup Guide

Step 1: Install Termux

Download Termux from F-Droid or Google Play

Step 2: Install Dependencies

pkg update && pkg upgrade
pkg install python git
pip install requests

Step 3: Download & Run Forwarder

# Download the script
curl -O https://raw.githubusercontent.com/123Lider/tempnum/main/android_forwarder.py

# Run with your API token
python android_forwarder.py --token YOUR_API_TOKEN_HERE
💡 Tip: Replace YOUR_API_TOKEN_HERE with the token shown when you registered your number!
